## 引用规则

- 保留标题、发布者、规范版本、链接和核验日期；
- 原创图示只表达经过复核的概念，不复制受版权保护的配图；
- 短引用必须服务于评论或解释，其余内容采用摘要；
- 来源变更后先复核课程结论，再更新 `lastReviewed`；
- 作者或发布机构只作为必要署名，不用于就业推荐。
