## 这是什么

AI 工程学习终端是一个静态、中文优先、Learning in Public 的教学网站。它从共同软件与 AI 基础开始，通过原创图示、最小实现和可重复实验逐步进入 Agent 工程及相关就业方向。

## 第一版范围

- 一张可按方向筛选的学习路线；
- 两节基础补给课和一节 `Workflow 与 Agent` 课程；
- 架构选择矩阵、网络流、上下文预算和 Agent Loop 等图示；
- 一个可切换故障路径的确定性 Agent Loop 组件；
- 七个抽象就业方向；
- 术语和公开来源索引。

## 不收集什么

本站没有账户、评论、远端数据库、广告、分析脚本或第三方追踪。

## 本地学习进度

模块完成状态使用带版本号的 `localStorage`，只保存模块 ID、状态和更新时间。当前数据很小，使用 IndexedDB 只会增加迁移和错误处理的复杂度。

这些记录不会离开浏览器，也不会跨设备同步。清除本站的浏览器站点数据即可删除。以后如果加入大段离线笔记、实验 Trace 或附件，再通过同一存储接口换成 IndexedDB，不需要重写课程组件。

## 公开内容边界

公开内容采用白名单，而不是从私有材料里“删几个词”后发布。站点不包含真实申请记录、特定岗位资料、个人联系方式、私有项目数据、凭据或本机路径。

规范、论文和技术文档的作者及发布机构只在来源页作必要署名。

## 内容状态

页面标记的“可学习”表示内容和交互已进入第一版验收；“计划中”只表示路线位置，不会伪装成已经完成的课程。
